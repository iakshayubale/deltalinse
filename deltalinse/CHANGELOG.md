## [1.0.0] - 2025-02-23

### Added
- Initial release of CI Diff Report
- Automatic regression detection (pass → fail)
- Fixed test detection (fail → pass)
- Flaky test detection
- Performance regression detection (20%+ slower)
- Intelligent failure clustering by error similarity
- Single self-contained HTML report generation
- PR comment markdown generation
- Drop-in CLI interface
- Full TypeScript implementation with strict mode
- Programmatic API for library usage
- Complete documentation (5 guides)
- Example test result files
- MIT License

### Features
- Zero configuration required
- Works with any JUnit XML format
- Supports all major CI/CD systems
- Responsive design (mobile-friendly)
- Print-to-PDF support
- Color-coded test status visualization
- Expandable test details
- Error message and stacktrace display

## How to Report Issues

Found a bug? Please open an issue on GitHub with:
1. Clear title describing the issue
2. Steps to reproduce
3. Expected vs actual behavior
4. Your environment (OS, Node version, etc.)
